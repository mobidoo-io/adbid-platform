===============================================
🚀 АЛЬТЕРНАТИВНЫЙ ДЕПЛОЙ (БЕЗ GITHUB)
===============================================

Поскольку репозиторий приватный, используем прямую загрузку.

ШАГ 1: НА ВАШЕМ КОМПЬЮТЕРЕ
---------------------------
1. Загрузите архив на файлообменник:
   - Откройте https://file.io или https://transfer.sh
   - Загрузите файл: adbid-deploy.tar.gz
   - Скопируйте ссылку для скачивания

ШАГ 2: НА СЕРВЕРЕ (в DigitalOcean Console)
-------------------------------------------
Выполните эти команды, заменив ССЫЛКА_НА_АРХИВ на вашу ссылку:

# Переход в папку
cd /var/www
rm -rf adbid
mkdir -p adbid

# Скачивание архива (замените ССЫЛКА_НА_АРХИВ)
wget -O adbid.tar.gz "ССЫЛКА_НА_АРХИВ"

# Или если используете file.io:
# curl -o adbid.tar.gz "ССЫЛКА_НА_АРХИВ"

# Распаковка
tar -xzf adbid.tar.gz -C adbid/
cd adbid

# Настройка Nginx
cat > /etc/nginx/sites-available/adbid << 'EOF'
server {
    listen 80;
    server_name adbid.me www.adbid.me 165.227.11.220;
    root /var/www/adbid/dist;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg|woff|woff2|ttf|eot)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
    
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_types text/plain text/css text/xml text/javascript application/javascript application/json;
}
EOF

# Активация сайта
ln -sf /etc/nginx/sites-available/adbid /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default

# Установка прав
chown -R www-data:www-data /var/www/adbid
chmod -R 755 /var/www/adbid

# Перезапуск Nginx
nginx -t && systemctl restart nginx

# Проверка
echo "========================================="
echo "✅ ГОТОВО! Сайт доступен по адресу:"
echo "   http://adbid.me"
echo "   http://165.227.11.220"
echo "========================================="

# SSL (опционально)
certbot --nginx -d adbid.me -d www.adbid.me --non-interactive --agree-tos -m admin@adbid.me

АЛЬТЕРНАТИВА 2: ПРЯМАЯ ЗАГРУЗКА ЧЕРЕЗ SCP
------------------------------------------
Если у вас есть SSH доступ с компьютера:

# На вашем компьютере:
scp adbid-deploy.tar.gz root@165.227.11.220:/tmp/

# На сервере:
cd /var/www
rm -rf adbid
mkdir adbid
tar -xzf /tmp/adbid-deploy.tar.gz -C adbid/
# Затем выполните настройку nginx как выше
