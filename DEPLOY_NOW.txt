ШАГ 2: СКОПИРУЙТЕ И ВЫПОЛНИТЕ ЭТИ КОМАНДЫ
------------------------------------------
После создания репозитория выполните в терминале:

git remote add origin https://github.com/ВАШ_USERNAME/adbid.git
git push -u origin main

(Замените ВАШ_USERNAME на ваш GitHub username)

ШАГ 3: КОМАНДЫ ДЛЯ СЕРВЕРА
---------------------------
После загрузки на GitHub, откройте DigitalOcean Console:
https://cloud.digitalocean.com/droplets

Найдите ваш droplet (165.227.11.220)
Нажмите "Access" → "Launch Droplet Console"

Скопируйте и выполните ВСЕ эти команды в консоли сервера:

=============== НАЧАЛО КОМАНД ДЛЯ СЕРВЕРА ===============

# 1. Установка пакетов
apt update && apt install -y git nginx certbot python3-certbot-nginx

# 2. Клонирование проекта
cd /var/www
rm -rf adbid.backup
mv adbid adbid.backup 2>/dev/null || true
git clone https://github.com/ВАШ_USERNAME/adbid.git
cd adbid

# 3. Настройка nginx
cat > /etc/nginx/sites-available/adbid << 'EOF'
server {
    listen 80;
    server_name adbid.me www.adbid.me 165.227.11.220;
    root /var/www/adbid/dist;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg|woff|woff2|ttf|eot)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
    
    gzip on;
    gzip_types text/plain text/css text/xml text/javascript application/javascript;
}
EOF

# 4. Активация сайта
ln -sf /etc/nginx/sites-available/adbid /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default

# 5. Установка прав
chown -R www-data:www-data /var/www/adbid
chmod -R 755 /var/www/adbid

# 6. Перезапуск nginx
nginx -t && systemctl restart nginx

# 7. Проверка
curl -I http://localhost
echo "Сайт должен быть доступен по: http://adbid.me"

# 8. SSL сертификат (ОПЦИОНАЛЬНО но рекомендуется)
certbot --nginx -d adbid.me -d www.adbid.me --non-interactive --agree-tos -m admin@adbid.me
