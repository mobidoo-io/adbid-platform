Adbid API Documentation

Version 2.0 • Last updated: August 2024

Introduction

Welcome to the Adbid API documentation. Our API allows you to programmatically manage your Meta Ads campaigns, access analytics, and leverage our AI-powered optimization features.

Base URL

ENDPOINT
https://api.adbid.ai/v2

Key Features

  • RESTful API design
  • JSON request and response format
  • OAuth 2.0 authentication
  • Comprehensive error handling
  • Real-time webhooks

Quick Start

Get up and running with the Adbid API in minutes.

Step 1: Get Your API Key

First, obtain your API key from the Adbid dashboard:

  1. Log in to your Adbid account
  2. Navigate to Settings → API Keys
  3. Click "Generate New Key"
  4. Copy and securely store your key

Step 2: Make Your First Request

BASH
curl -X GET https://api.adbid.ai/v2/campaigns \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json"
JAVASCRIPT
const response = await fetch('https://api.adbid.ai/v2/campaigns', { headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' } }); const campaigns = await response.json(); console.log(campaigns);
PYTHON
import requests response = requests.get( 'https://api.adbid.ai/v2/campaigns', headers={ 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' } ) campaigns = response.json() print(campaigns)

Authentication

The Adbid API uses API keys to authenticate requests. Include your API key in the Authorization header:

HEADER
Authorization: Bearer YOUR_API_KEY

OAuth 2.0 Flow

For third-party integrations, we support OAuth 2.0 authorization code flow:

POST /oauth/authorize
Parameter Type Description
client_id string Required Your application's client ID
redirect_uri string Required The callback URL
scope string Optional Requested permissions

Campaigns

Manage your advertising campaigns programmatically.

List All Campaigns

GET /campaigns

Create Campaign

POST /campaigns
JSON
{ "name": "Summer Sale Campaign", "objective": "CONVERSIONS", "budget": { "amount": 1000, "currency": "USD", "type": "DAILY" }, "targeting": { "locations": ["US", "CA"], "age_min": 18, "age_max": 65, "interests": ["shopping", "fashion"] } }

Update Campaign

PUT /campaigns/{campaign_id}

Delete Campaign

DELETE /campaigns/{campaign_id}

Rate Limits

API rate limits vary by plan:

Plan Requests/Hour Burst Rate
Starter 1,000 50/min
Growth 10,000 200/min
Scale 50,000 1,000/min
Enterprise Unlimited Custom

Rate limit information is included in response headers:

  • X-RateLimit-Limit - Request limit per hour
  • X-RateLimit-Remaining - Remaining requests
  • X-RateLimit-Reset - Time when limit resets
